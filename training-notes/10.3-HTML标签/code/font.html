HTML标签 - font标签(字体效果)

font 语法

<font> 规定文本的字体、字体尺寸、字体颜色。

<font size="字体大小" color="字体颜色" face="字体类型">要设置的文字</font>

size: 12,13 ...

color: red,blue,green,yellow 或者 颜色值#333

face: "黑体" "微软雅黑" "宋体"

Eg:

    中村天风强调有意识地加以注意的重要性，他认为“ 只有有意注意的人生才有意义 ”。我们的注意力是有限的，总是集中意识注意一个事情是困难的，但是，如果用心就能逐步养成有意注意的习惯，就能抓住事物的本质和核心，具备准确的判断能力。
